# Usage
## Requirements
Java 8+
## Start Server
```
java -jar target/server.jar
```
## Start Client
```
java -jar target/client.jar
```

# Building
## Requirements
Java 8+
Maven 3+
## Command
```
mvn clean compile package
```
This will generate 2 jar files in the target directory:
- server.jar
- client.jar
